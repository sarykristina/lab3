# Распознавание рукописных цифр с помощью CNN

## О проекте

Задача многоклассовой классификации рукописных цифр (0–9) с использованием свёрточной нейронной сети (CNN). В проекте исследуется влияние аугментации данных на качество модели, а также визуализируются feature maps для понимания того, что именно учит сеть.

## Данные

- Датасет: `train.csv` / `test.csv` (формат Kaggle MNIST)
- Входные данные: изображения 28×28 пикселей, нормализованные в [0, 1]
- Целевая переменная: цифра от 0 до 9

## Что сделано

- Реализация кастомного `Dataset` с поддержкой аугментации (повороты, сдвиги)
- Построение CNN архитектуры: Conv2d → ReLU → MaxPool2d (два блока) → Linear → Dropout
- Обучение двух моделей: без аугментации и с аугментацией
- Визуализация feature maps на каждом слое сети
- Сравнение точности моделей на чистых и аугментированных данных
- Тест моделей на собственных фотографиях рукописных цифр

## Результат

Модель с аугментацией показывает более высокую устойчивость к искажениям входных данных.

## Стек

- Python, PyTorch, Pandas, NumPy, Scikit-learn, Matplotlib
