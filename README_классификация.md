# Классификация грибов с помощью нейронных сетей

## О проекте

Задача бинарной классификации: определить, является ли гриб съедобным или ядовитым, на основе его характеристик.

## Данные

- Датасет: `mushrooms.csv`
- Признаки: категориальные характеристики гриба (форма шляпки, цвет, запах и др.)
- Целевая переменная: `class` — съедобный (`e`) или ядовитый (`p`)

## Что сделано

- Разведочный анализ данных (EDA): распределение классов
- Кодирование категориальных признаков через One-Hot Encoding
- Отбор признаков по взаимной информации (порог 0.05)
- Обучение и сравнение двух моделей на PyTorch:
  - Однослойный перцептрон
  - Многослойный перцептрон (MLP) с ReLU-активацией
- Оценка моделей: accuracy, матрица ошибок, classification report

## Стек

- Python, PyTorch, Pandas, NumPy, Scikit-learn, Matplotlib
