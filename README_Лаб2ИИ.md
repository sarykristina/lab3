# Распознавание рукописных цифр (MNIST)

## О проекте

Задача многоклассовой классификации: распознавание рукописных цифр от 0 до 9 по пиксельным данным.

## Данные

- Датасет: `train.csv` / `test.csv` (формат Kaggle MNIST)
- Входные данные: изображения 28×28 пикселей (784 признака), нормализованные в диапазон [0, 1]
- Целевая переменная: цифра от 0 до 9

## Что сделано

- Загрузка и предобработка данных
- Реализация кастомного класса `Dataset` для PyTorch
- Поддержка двух режимов входных данных: плоский вектор и изображение (1×28×28)
- Обучение нейронной сети с функцией потерь CrossEntropy и оптимизатором Adam
- Оценка качества на валидационной выборке (90/10 split)

## Стек

- Python, PyTorch, Pandas, NumPy, Scikit-learn, Matplotlib
